# PDF 萬能工具箱

一個瀏覽器端 PDF 處理工具，可在本機直接上傳 PDF、拆成單頁、選取頁面、合併，並輸出為 PDF 或 JPG。

## 功能

- 上傳或拖放多個 PDF 檔案
- 自動將 PDF 拆成單頁縮圖
- 預設選取全部頁面
- 依單數頁、偶數頁、全部或自訂頁碼範圍快速選取
- 手動移動頁面順序
- 移除單一頁面
- 一鍵清空工作區
- 合併選取頁面或全部頁面
- 保持單頁文件，不合併輸出
- 輸出為 PDF 或 JPG

## 使用流程

1. 開啟 `index.html`。
2. 點選「上傳 PDF」，或將 PDF 拖放到主工作區。
3. 系統會自動將 PDF 分割成單頁縮圖，並預設選取全部頁面。
4. 依需求調整頁面：
   - 選取單數頁、偶數頁、全部頁面
   - 輸入頁碼範圍，例如 `1, 3-5`
   - 點選縮圖切換選取狀態
   - 使用左右箭頭調整順序
   - 使用刪除按鈕移除頁面
5. 選擇合併方式：
   - 「合併選取頁面」：只合併目前選取的頁面
   - 「全部頁面合併」：依目前排序合併所有頁面
   - 「保持單頁文件」：保留選取頁面為個別文件
6. 選擇輸出格式：
   - `PDF 檔`
   - `JPG 檔`
7. 點選「輸出檔案」下載結果。

## 輸出行為

| 合併方式 | PDF 輸出 | JPG 輸出 |
|---|---|---|
| 合併選取頁面 | 產生一個合併後的 PDF | 每頁輸出一個 JPG |
| 全部頁面合併 | 產生一個合併後的 PDF | 每頁輸出一個 JPG |
| 保持單頁文件 | 每頁輸出一個 PDF | 每頁輸出一個 JPG |

## 隱私與安全

本工具在瀏覽器端處理 PDF。檔案內容不會主動上傳到伺服器。

注意：目前頁面使用 CDN 載入前端函式庫，因此需要網路連線取得依賴套件。若要完全離線使用，可將相關 JavaScript 函式庫下載到專案內並改成本地引用。

## 技術

- HTML
- Tailwind CSS CDN
- Lucide Icons CDN
- PDF-LIB
- PDF.js

## 本地使用

這是純前端單頁工具，不需要建置流程。

直接用瀏覽器開啟：

```text
index.html
```

或使用任意靜態檔案伺服器開啟本專案目錄。

## 目前限制

- 加密或受保護的 PDF 可能無法處理。
- JPG 輸出會逐頁下載，瀏覽器可能對大量下載跳出提示。
- 大型 PDF 會占用較多瀏覽器記憶體。
- 輸出 JPG 時，每個 PDF 頁面會輸出為一張 JPG，不會打包成 ZIP。
